Call doctor

The main aim is to provide better platform between Doctor and Patient. This is the application which will be useful for the people who want immediate communication with the doctors. The Users will able to select the doctors based on their problem, Can Make appointment,Can maintain their favourite doctors list,Can chat with doctor either text or video after paying money through online and also can save their previous appointment list in their dashboard. The appointment can be accepted, if the doctor working hours feasible for that time. Doctors can be certified before joining to this website through trusted agencies. Based on this the information the users can select doctor for treatment. Another feature also available to the users to select doctors i.e based on City, Specialist. The doctors verification process will be done by certified authority. The medical tools – BMI Calculator (Body Mass Index), BMR Calculator (Basal Medical rate), Total Body water,Risk Calculator.

Clustered information retrieval using image query.

In this paper the concept is like searching some information using images in search engines. The process is uploading an image, which will go under various search techniques to give an acceptance result. Here the search technique will detect the edge of the image and also the color of the image, N-process and finally it will calculate the similarity of two images (one which will uploaded and another image which already in server where it will check for similarity).So the above process will be done using color coherence vector matrix. Finally the use can get correct information about what he uploaded.
